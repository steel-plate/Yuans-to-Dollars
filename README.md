# Yuans to Dollars
一个简单的Obsidian插件，用于把连续输入的两个“￥”符号替换成连续的两个“$”符号，并把光标移动到美元符号中间，如果配合自动切换输入法插件，可以获得极其顺滑的中文语境公式输入流程。除此之外没有其他功能，方便与其他任何插件相配合
## 使用方法
1. 在Obsidian中打开设置，进入“第三方插件”选项页，选择关闭安全模式
2. 在社区插件市场中搜索“BRAT”安装并启动
3. 关闭设置页面，打开BRAT选项，选择“Add a beta plugin for testing (with or without version)”
4. 在弹出的页面中复制本项目地址URL，并选择唯一发行版本进行安装
5. 安装推荐的输入法自动切换软件并按照其说明调整（可选）
6. 使用中文输入法输入“￥￥”并享受
## 推荐的自动切换输入法插件
- IMSwitch in Math Block
